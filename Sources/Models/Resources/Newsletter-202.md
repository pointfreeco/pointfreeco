Today we are officially releasing the "[Point-Free Way](/the-way)": a collection of skill documents that can be used in a variety of AI tools to best make use of our open source libraries and employ the patterns we've discussed on Point-Free.

For the past 8 years we have focused on building applications with a few key things in mind: simple, composable units that can be pieced together, prefer value types over reference types when appropriate, control dependencies on the outside world, and make _everything_ testable. The Point-Free Way brings these ideas together so that they can be applied in day-to-day development of real world code bases. 

Get started with 3 simple commands:

```sh
brew install pointfreeco/tap/pfw
pfw login
pfw install
```

This will install the skill documents into whichever AI tool you are using (Claude, Codex, Gemini, etc.). Further, we are constantly updating these skills. Whenever we answer a question from the community we will encode its answer into the skills, and whenever we add new features to our libraries we will update the skills. All updates are just a `pfw install` away!

And this is just the beginning. We have a lot more planned, so give the tool a spin today and join our [Slack](/slack-invite) to provide any feedback!
